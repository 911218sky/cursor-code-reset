cursor-code-reset

This extension (cursor-code-reset) backs up and refreshes CSP marker UUIDs in Cursor’s main.js.

Close other Cursor windows before running this if possible; reload the window when done. When you choose Start reset workflow, the extension reads main.js and shows the current CSP1 / CSP4 IDs in a confirmation dialog before applying changes.

Use the status bar shortcut “cursor-code-reset” on the right. Follow Cursor’s terms of use.
